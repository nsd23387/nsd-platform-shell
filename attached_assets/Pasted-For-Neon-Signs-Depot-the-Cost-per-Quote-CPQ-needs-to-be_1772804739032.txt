3. Exact Quote Definition (Critical)

Quotes should include only revenue-intent events:

custom_quote_submit
wholesale_quote_submit
channel_letter_quote_submit

Do NOT include:

contact forms

chats

phone calls

Otherwise CPQ becomes distorted.

4. Implementation in Your Data Model

Since your canonical source is:

analytics.raw_web_events

The quote count is:

event_type IN (
 'custom_quote_submit',
 'wholesale_quote_submit',
 'channel_letter_quote_submit'
)
5. SQL Implementation
Cost per Quote (Google Ads)
SELECT
SUM(spend) / NULLIF(COUNT(q.event_id),0) AS cost_per_quote
FROM analytics.raw_google_ads ads
LEFT JOIN analytics.raw_web_events q
ON ads.gclid = q.gclid
WHERE q.event_type IN (
 'custom_quote_submit',
 'wholesale_quote_submit',
 'channel_letter_quote_submit'
);
Cost per Quote (All Channels)
SELECT
channel,
SUM(spend) / COUNT(*) AS cost_per_quote
FROM conversion_events
GROUP BY channel;
